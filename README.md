# Mystic Leapfall

Mystic Leapfall is a 2D side-scrolling platformer inspired by classic games like Super Mario Bros. The game blends nostalgic platforming action with a magical fantasy atmosphere, delivering colorful worlds, smooth controls, and exciting challenges.

---

## 🎮 Game Overview

In Mystic Leapfall, players control a brave young hero exploring enchanted realms filled with floating platforms, tricky enemies, hidden passages, and powerful bosses. Your mission is to jump, dodge, collect, and survive as you travel across mystical lands.

Each level introduces new obstacles and gameplay mechanics designed to test your timing, reflexes, and strategy.

---

## ✨ Features

- Classic 2D side-scrolling platform gameplay  
- Responsive and smooth controls  
- Multiple themed environments:
  - Enchanted Forest  
  - Crystal Caverns  
  - Sky Fortress  
  - Lava Depths  
- Unique enemy types and boss encounters  
- Special abilities:
  - Double Jump  
  - Dash Boost  
  - Magic Shield  
- Hidden collectibles and secret paths  
- Gradually increasing difficulty  

---

## 🕹️ Controls

**Keyboard (PC):**

- Move Left: A / Left Arrow  
- Move Right: D / Right Arrow  
- Jump: Space  
- Dash: Shift  
- Pause: Esc  

---

## 🚀 How to Play

1. Launch the game.  
2. Choose a level.  
3. Navigate platforms and avoid enemies.  
4. Collect coins and power-ups.  
5. Reach the goal to complete the stage.  

Be careful — falling or touching enemies without protection will cost you a life!

---

## 🛠️ Installation

1. Download the game files.  
2. Extract the folder (if compressed).  
3. Run the executable file to start playing.
